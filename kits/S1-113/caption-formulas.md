# Caption formulas (paste app details, get captions)

Use these with Claude — prompt: "App: {desc}. Target user: {who}. Write 6 screenshot captions
using the framework: hook, proof, feature, feature, objection-killer, CTA. <=7 words each."

Hook: "Finally, {outcome} without {pain}."  e.g. "Finally, budgets that update themselves."
Proof: "Loved by {N} {users}." / "4.9★ — {quote}."
Feature: "{Verb} {noun} in seconds." e.g. "Split bills in seconds."
Objection: "Free to start. No sign-up." / "Private. Your data stays on-device."
CTA: "Start {outcome} today — free."
