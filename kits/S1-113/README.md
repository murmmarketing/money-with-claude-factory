installs — they feel every % of conversion. Find them on r/iOSProgramming, Indie Hackers,
Twitter/X #buildinpublic, and the "recently updated" App Store lists.

**Offer:** "6 conversion-optimized screenshots (captions + layout + device frames) for your app,
in 4 days. $349. Free redesign of your first screenshot so you can see the lift."

**Price:** $200–800/app. Add $99/mo "seasonal refresh" retainer (holiday/feature updates).

## Launch in 60 minutes
1. Read `screenshot-framework.md` — the 6-slot template (hook → proof → features → CTA).
2. Pick ONE real app you like on the store, redesign its first screenshot's *copy + layout* using

3. Post the teardown publicly (Reddit/X) tagging the dev — that IS your marketing (marketing.md).
4. **[YOUR MOVE]** set up payment + a Figma/Canva file to actually produce the visuals.
