# The 6-slot App Store screenshot framework

Stores show the first 2–3 in search. Front-load the hook.

| # | Job | Caption pattern | Visual |
|---|-----|-----------------|--------|
| 1 | **Hook** — the #1 benefit | "Finally, {outcome} without {pain}" | Big text top, app UI below in device frame |
| 2 | **Proof** | "Rated 4.9 by 12,000 {users}" / testimonial | Stars + a real review card |
| 3 | **Core feature** | "{Feature} that {benefit}" | UI zoomed to the feature, arrow/highlight |
| 4 | **Secondary feature** | "Also: {benefit}" | Split screen or second UI |
| 5 | **Objection killer** | "Free to start · No account needed" | Icon row of trust points |
| 6 | **CTA** | "Download — your {outcome} starts today" | Bold, brand color, app icon |

Rules: one idea per screenshot · caption readable at thumbnail size · consistent font+color ·
device frames current-gen · localize captions for top 3 markets (upsell).
