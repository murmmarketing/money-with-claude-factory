# HaulHQ — OG / Social Image Spec

Spec for the Open Graph / Twitter card image, and for the dynamic per-haul OG images
generated by the existing `/api/og` route. Colors, fonts and voice come from `brand.json`.

## Canvas & format
- **Size:** 1200 × 630 px (1.91:1). Safe area: keep all text/logo inside a 60px inner margin.
- **Formats:** PNG (static default OG at `og-default.png`) and dynamically rendered per-haul cards via `/api/og` (edge, `@vercel/og`).
- **Weight:** target < 300 KB for the static card.
- **Twitter card type:** `summary_large_image`.

## Theme
Dark trader-terminal by default (matches the product's dark-first UI). A light variant may
be produced for print/light contexts, but the dark card is primary.

- Background: `#0B0F10` (dark `bg`).
- A faint terminal grid: 1px lines in `#151B1C` on a 40px grid, ~8% opacity — texture only, never competes with text.
- Optional hairline border inset 24px in `#28312F`.

## Layout (static default: og-default.png)
Left-aligned, top-to-bottom, like a terminal readout.

1. **Logo lockup** — top-left, `logo.svg` at ~360px wide (the self-contained panel or just the mark + "HaulHQ" wordmark).
2. **Headline** — Space Grotesk 700, ~76px, `#F5F7F5`, letter-spacing -0.02em, max 2 lines:
   > Know what your haul REALLY costs.
   With "REALLY" or the money figure emphasized in lime `#B8FF35`.
3. **Subhead** — Space Grotesk 500, ~34px, `#9AA7A2`, 1 line:
   > Landed cost + cheapest-agent compare for reps buyers. Free.
4. **Hero money chip** (bottom-left) — the signature move: a lime pill.
   - Pill fill `#B8FF35`, radius 12px, text `#071008` (accentInk), Space Grotesk 600 ~40px, tabular-nums:
     > You'd save $37 with Agent X
5. **Footer domain** — bottom-right, `#9AA7A2`, Space Grotesk 500 ~26px: value of `NEXT_PUBLIC_SITE_URL` (default `haulhq.app`).

## Layout (dynamic per-haul card: /api/og?type=haul)
For shareable public haul pages `/h/[slug]` — this is the viral/SEO surface.

- Top-left: HaulHQ mark (small, ~140px) + "shared haul" label in `#9AA7A2`.
- Big **haul title** (from `hauls.title`) — Space Grotesk 700 ~64px, `#F5F7F5`, truncate to 2 lines.
- A compact stat row (mono numeric stack, tabular-nums, `#F5F7F5`) rendered as 3 cells on `#151B1C` cards with `#28312F` borders:
  - `ITEMS` — item count
  - `SUBTOTAL` — e.g. ¥3,840 → converted currency
  - `LANDED` — landed total in the haul's currency, the number in lime `#B8FF35`
- Destination flag/label (e.g. "→ EU · VAT included") in `#9AA7A2`.
- Optional: up to 3 QC/cover thumbnails from `haul_items.qc_photo_url` / `hauls.cover_image_url`, right side, 1px `#28312F` border, radius 10px. Degrade gracefully to no-thumbs.
- Footer-right: domain.

### /api/og query params (contract for the app thread)
- `type` = `default` | `haul` (default `default`)
- `title` — haul title (haul type)
- `landed` — formatted landed total string (e.g. `$611`)
- `items` — integer count
- `dest` — destination country code/label
- `save` — optional savings line for the lime chip (e.g. `Save $37 vs Agent Y`)
All params must be URL-encoded and length-clamped server-side; missing params fall back to the static default composition.

## Typography rules
- Font: **Space Grotesk** (display + text), matching `brand.json.fontKey = "grotesk"`.
  In `@vercel/og`, load the .ttf via `fetch` in the route and pass to `fonts` (weights 500/600/700).
- All money/weight/count figures use `font-variant-numeric: tabular-nums`.
- Never use emoji. Tone is blunt and insider (see `brand.json.voice`).

## Color usage (from brand.json)

| Background | `#0B0F10` |
| Card surface | `#151B1C` |
| Borders / grid | `#28312F` |

| Risk / overpay / tax-hit accents | `#FF5C5C` |

Contrast: every text/background pair above is WCAG-AA-verified in `brand.json.palette.wcag_aa_pairs`.

## Do / Don't
- **Do** make a real number the visual hero on every card (savings or landed total).
- **Do** keep it terminal-clean: flat fills, hairline borders, no drop shadows or gradients.
- **Don't** use lime for large body text or on light backgrounds (fails contrast — use `#15723A` in light theme).
- **Don't** center everything; left-aligned readout reads as a tool, not a brochure.
